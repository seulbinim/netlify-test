# 멋쟁이사자처럼 프론트엔드 UI 프로젝트 12기 엄현욱 회고록

## 프로젝트 개요

- 기간: 2025.3.14 ~ 2025.3.25
- 참고한 사이트: [티빙](https://www.tving.com/onboarding)
- 최종 배포 페이지 : [타잉]()
- 사용 스택 : HTML5 / tailwind CSS 4.0

## 프로젝트를 통해 얻은 것

- 팀원들과 같이 저장소를 관리하고, issue를 생성하고, pull&request를 진행하고, 서로 검토해주고 merge하는 과정에서 개발자로서 팀 업무를 진행하는 방식에 대해 어렴풋이나마 알 수 있어서 좋았다.
- 컴포넌트 베이스로 작업을 하는 방식에 익숙해질 수 있었다.
- 새롭게 출시된 tailwind CSS 4.0에 익숙해질 수 있었다.
- 수업시간 동안 배웠던 html과 css를 적절하게 써가며 화면을 구성하는 방법에 익숙해졌다.

## 어려웠던 점

- 반응형으로 페이지를 구성할 때, breakpoint별로 매끄럽게 이어지도록 레이아웃을 구성하는게 힘들었다. 특히, 화면의 사이즤를 키우고 줄일 때, 브레이크포인트를 기준으로 width나 height가 뚝뚝 끊어지는 느낌이 드는 부분을 세부적으로 조정해주는게 어려웠다.
- grid로 페이지 레이아웃을 구성하는 것이 생각보다 까다로웠다.
- javascript를 사용하지 않고 html과 css만으로 화면을 비슷하게 구현을 하는 것이 오히려 더 까다롭게 느껴졌다.

## 개선할 점

- javascript를 배운 뒤, 화면에 여러 동적인 기능을 추가하면 좋을 것 같다.
